# マインスイーパー 仕様書

## 画面サイズ

- 固定サイズ: 800 x 600 px

## 画面構成

### 1. 初期画面（難易度選択）

- タイトル「マインスイーパー」を表示
- 難易度選択ボタンを3つ縦に並べる
  - 初級
  - 中級
  - 上級

### 2. ゲーム画面

#### レイアウト

```
+----------------------------------+
|  ヘッダー                         |
|  [残り地雷数] [リスタート] [タイマー]|
+----------------------------------+
|                                  |
|          盤面                     |
|                                  |
+----------------------------------+
```

#### ヘッダー

- 残り地雷数カウンター（左側）
- リスタートボタン（中央）
- タイマー（右側）

#### 盤面

| 難易度 | 盤面サイズ | 地雷数 | マスサイズ（目安） |
|--------|-----------|--------|------------------|
| 初級   | 9x9       | 10     | 画面に収まるよう調整 |
| 中級   | 16x16     | 40     | 画面に収まるよう調整 |
| 上級   | 30x16     | 99     | 画面に収まるよう調整 |

※ 上級（30x16）が最大サイズ。800x600からヘッダー分を引いた領域に収まるようマスサイズを算出する。

## マスの表示

シンプルな図形で描画する（後から画像に差し替え可能な設計）。

| 状態 | 表示 |
|------|------|
| 未開放 | グレーの四角 |
| 開放済み（数字） | 明るい四角 + 数字（1-8） |
| 開放済み（空白） | 明るい四角 |
| フラグ | グレーの四角 + フラグマーク |
| 地雷（ゲーム終了時） | 地雷マーク |

## 操作

| 操作 | アクション |
|------|-----------|
| 左クリック | マスを開く |
| 右クリック | フラグを立てる/外す |
| 両クリック | 数字マスの周囲のフラグ数が一致した場合、周囲の未開放マスを一括で開く。フラグの位置が間違っていた場合は敗北。 |

## タイマー

- 開始タイミング: 最初のマスをクリックした時
- 表示形式: 秒数（整数）
- 上限: 999秒（それ以上は999で固定表示）

## 残り地雷数カウンター

- 計算: 総地雷数 - フラグを立てた数
- マイナス表示: あり（フラグを地雷数より多く立てた場合）

## リスタートボタン

- 動作: 同じ難易度で新しいゲームを開始
- 盤面と地雷配置はランダムに再生成

## ゲーム終了

### 勝利条件

- 地雷以外の全てのマスを開いた時

### 敗北条件

- 地雷のあるマスを開いた時

### 終了時の動作

- 敗北時: 全ての地雷マスを開いて表示
- 勝利時: 全ての地雷マスに旗を表示
- マスのクリック操作を無効化
- リスタートボタンのみ操作可能

## 最初のクリック

- 最初にクリックしたマスには地雷を配置しない
- 地雷配置は最初のクリック後に行う
